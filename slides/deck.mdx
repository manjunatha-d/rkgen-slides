# rkgen

---

# What's rkgen?

- A micro-generator framework for Polaris UI
- Scaffolding tools
    - Project level - CRA, Yeoman, etc
    - Parts of projects - Hygen, PlopJS, etc

# The WHYs


<Steps>

### Speed up UI development e.g. adding new snappables
### Ensure consistency in code and UX
### Limit the number of things a backend engineer needs to learn

</Steps>

---

# Overview

## A micro-generator framework should be able to:

<ul>
  <Steps>
    <li>Take input from user</li>
    <li>Add new files</li>
    <li>Modify existing files</li>
  </Steps>
</ul>

---

# Tools evaluation

## PlopJS, Hygen

- User inputs - Prompts using inquirer.js
- Modify existing files - regex based modifications
- Add new files - templating engines like Handlebars, EJS.

---

# Why homegrown solution?

<ul>
  <Steps>
    <li>AST based modifications give more confidence</li>
    <li>No need to learn templating engines</li>
  </Steps>
</ul>
